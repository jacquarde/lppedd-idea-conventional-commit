0.24.0 (03/01/2025)

Thirty-eighth alpha release.

This release contains the following fixes and refactorings:

  • Do not swallow ProcessCanceledException (#143)
  • Synchronize the registration of commit inspections (#146)
  • Use the IntelliJ Platform Plugin v2 (89f6eb7)
  • Replace some usages of deprecated APIs (979a8a9)
  • Attempt to further alleviate WSL/Gateway issues (dc77708 and 8530dab)

Note that the minimum required IDEA version is now 233.11799.
