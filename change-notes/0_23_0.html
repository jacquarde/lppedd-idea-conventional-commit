0.23.0 (08/04/2024)

Thirty-sixth alpha release.

This release contains the following enhancements and fixes:

  • Update Chinese translation
  • Allow specifying a schema in conventionalcommit.json files
  • Improve the quick documentation's presentation
  • Fix the Reformat Commit Message action
  • Set the execution thread for all actions
  • Remove redundant "file" suffix for the Conventional Commit file type
  • Remove most usages of deprecated signatures
  • Move the minimum supported IDEA version to 2022.3
  • Move the maximum supported IDEA version to 2024.3
